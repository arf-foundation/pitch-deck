# ARF Pitch Deck – Agentic Reliability Framework

This repository contains the 3‑minute pitch deck for the **Agentic Reliability Framework (ARF)**, designed for the *Pioneering Minds AI Pitch Masterclass*.

## 📁 Contents
- `slide1.html` – Title & Tagline  
- `slide2.html` – The Problem (with references)  
- `slide3.html` – Current Limitations  
- `slide4.html` – The ARF Solution (with Mermaid diagram)  
- `slide5.html` – How It Works (pipeline + code blocks)  
- `slide6.html` – Key Capabilities & Differentiators  
- `slide7.html` – Traction & Live Demo (with QR codes)  
- `slide8.html` – Team & Credibility  
- `slide9.html` – Vision & Roadmap  
- `slide10.html` – Call to Engage  

## 🖨️ Usage
Open any `.html` file in a browser to view the slide.  
For a combined deck, open `all-slides.html` (if created) or print each slide to PDF and merge.

## 🎯 Purpose
- Non‑fundraising product pitch (3 minutes)  
- Focus: problem, solution, technical depth, traction, team
